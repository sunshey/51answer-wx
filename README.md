项目介绍
---
一款在线搜索书籍、查找答案，辅助学生提高学习成绩的小程序 
1. 数据源来自于线下采集，购买书籍拍照上传，用户自发上传
2. 界面使用小程序原生组件设计，暂无扩展组件，界面样式使用了部分伪类选择器，比如.xx::after{...}
###### 说明：这款小程序是我用来入门的项目，暂无上线小程序商店
